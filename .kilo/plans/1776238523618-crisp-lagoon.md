## Bonus: Fix Sass @import Deprecation

Sass `@import` is deprecated in Dart Sass 3.0. Must migrate to `@use`.

### Changes to `assets/css/main.scss`:

Replace `@import` with `@use`:

```scss
/* Before (deprecated) */
@import "abstracts/variables";

/* After (modern) */
@use "abstracts/variables" as *;
```

### Files to update:
- `abstracts/variables` - vars used globally, need `as *`
- `abstracts/mixins` - mixins, need `as *`  
- `base/*` - if vars used in these, add `@use` first
- `layout/*` - same
- `components/*` - same
- `sections/*` - same
- `blog` - same

### Migration order:
1. Add `@use` statements at top
2. Remove deprecated `@import` 
3. Test build
